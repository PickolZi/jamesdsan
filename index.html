Projects

Spotify Music Downloader

Using the light weight framework Flask in the backend, I was able to develop the Spotify Music Downloader. This web application allows the client to browse any Spotify User's profile and download the playlist onto their computer.

Building this web application taught me a lot in developing fullstack applications.

Using Flask for the backend, plain HTML and CSS for the frontend, fetching data from Spotfiy's Developer API, using Celery and redis as a task queue and quick cache database to store and process the queues for the client's downloads, and lastly Docker in order to easily containerize and run different processes by simply running "docker compose up".
