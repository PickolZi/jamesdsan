Spotify Music Downloader

Spotify Music Downloader is a flask web application I created in order to download any or all spotify songs from any Spotify user's playlists.

It works by taking in a Spotify user's id you see when you visit their user page on Spotify and displays all of their visible playlists. From there you are able to select which songs you want and download them, quick and easy!

If you want a more technical overview of how this project works, click the github icon down below to view the source code.
